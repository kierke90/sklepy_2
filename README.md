# sklepy_2

Repozytorium zawiera notebook Jupyter `sklepy_2.ipynb`.

## Wymagania

- Python 3.10+ (zalecane)
- Jupyter Notebook lub JupyterLab

## Uruchomienie

1. Sklonuj repozytorium:
   `git clone https://github.com/kierke90/sklepy_2.git`
2. Przejdz do katalogu projektu:
   `cd sklepy_2`
3. Uruchom Jupyter:
   `jupyter notebook`
4. Otworz plik `sklepy_2.ipynb`.
